# Telegram Bot Service
Telegram Bot Service

# Methods
  #### sendMessageTo
  Use this method to send message to given chat.
  ```php
  $bot = new TelegramBotService($token);
  $result = $bot->sendMessageTo($chat_id, $message);
  ```
